Envia para o twitter os posts publicados, usando OAuth da sua App.

== Description ==

O Tweet myPost, avisa na sua conta do twitter quando você publica uma nova postagem, usando a Twitter OAuth registrada por você, assim seu posta sempre sairá com o "via Seu Blog".

== Installation ==

1. Envie a pasta `tweet_mypost` para o diretório `/wp-content/plugins/`
2. Ative o plugin indo em 'Plugins' no menu WordPress
3. Clique em 'Tweet myPost' no menu WordPress, e configure.

Mais informações em [http://reitor.org/wp-plugins/tweet_mypost/](http://reitor.org/wp-plugins/tweet_mypost/)
